# The Carpentry Collective — Netlify starter

This is a simple static site (HTML/CSS) ready for Netlify.

## Deploy
1. Create a GitHub repo and upload these files.
2. In Netlify: Add new site → Import from Git → select the repo → Deploy.
3. In Netlify: Domain settings → add carpentry.org.au and follow the DNS instructions.

## Forms
The volunteer form uses Netlify Forms.
After your first deploy, submit the form once, then check:
Netlify → Site → Forms
